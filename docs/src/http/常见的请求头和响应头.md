## HTTP 常见请求头 & 响应头（分类速记）

### 请求头（Request Headers）
- **内容协商**：
  - `Accept`（`text/html`、`application/json`）
  - `Accept-Language`（`zh-CN`、`en-US`）
  - `Accept-Encoding`（`gzip`、`br`）
  - `Accept-Charset`（`utf-8`）
- **身份与安全**：
  - `Authorization`（`Bearer <token>`、Basic Auth）
  - `Cookie`
- **请求体信息**：
  - `Content-Type`（`application/json`、`multipart/form-data`、`application/x-www-form-urlencoded`）
  - `Content-Length`
- **缓存相关**：
  - `If-Modified-Since`
  - `If-None-Match`
- **连接与路由**：
  - `Host`（目标主机，HTTP/1.1 必须）
  - `Connection`（`keep-alive`、`close`）
  - `Referer`（来源页面 URL）
  - `User-Agent`（浏览器或客户端信息）
  - `Date`（请求发起时间）

### 响应头（Response Headers）
- **内容描述**：
  - `Content-Type`
  - `Content-Encoding`
  - `Content-Length`
- **缓存控制**：
  - `Cache-Control`（`no-cache`、`max-age`）
  - `Expires`
  - `ETag`
  - `Last-Modified`
- **重定向 & 跳转**：
  - `Location`
  - `Refresh`
- **身份 & 状态维护**：
  - `Set-Cookie`
  - `WWW-Authenticate`
- **跨域相关**：
  - `Access-Control-Allow-Origin`
  - `Access-Control-Allow-Methods`
  - `Access-Control-Allow-Headers`
- **服务器信息**：
  - `Server`（服务器软件信息）
  - `Date`（响应时间）
