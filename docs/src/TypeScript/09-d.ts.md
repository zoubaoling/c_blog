## *.d.ts
声明文件
1. export 导出
```ts
declare module 'lodash' {
  type Comparator = (index: number) => string
  // 使用export导出，需要使用importd导入
  export const Com: Comparator
  // 不使用export，就全局使用，文件中不需要import
  interface GlobalUtilType: () => void
}
```
2. 函数类型和声明类型
```ts
declare module 'wizard' {
  // getComparator<T> -> 声明时直接显式传入T类型getComparator<String>
  export getComparator<T> = (arr: T[]) => T
  // <T>() -> 声明时不需要显式传入 T类型，是推断而来。const index:getIndex = ...
  export const getIndex = <T>(arr: T[]) => number
}
```

3. 扩展三方库声明文件
- TS会自动将同名的接口、类、命名空间合并
- 扩展不是重新定义，是扩展现有类型，不是创建新的导出,不需要export
- 声明全新的接口，需要export
```ts
// types/lodash-extensions.d.ts
declare module 'lodash' {
  // 扩展 LoDashStatic 接口，不要需要export
  interface LoDashStatic {
    customMethod<T>(array: T[]): T[];
    myCustomFilter<T>(array: T[], predicate: (item: T) => boolean): T[];
  }
  
  // 扩展数组原型
  interface LoDashImplicitWrapper<T> {
    myCustomMap<U>(iteratee: (value: T) => U): LoDashImplicitWrapper<U[]>;
  }
}
```
