## 两栏/三栏布局实现（面试速记版）

> 问题关键词：左右/中间自适应、经典布局实现方式、兼容性选型。回答时先给出主推方案（通常是 `flex`），再补充 1~2 种老方案与注意点。

### 快速选择指南
- **现代项目**：`flex`/`grid`，代码最少、响应式最好。
- **兼容 IE8**：浮动或绝对定位；记得用 BFC/`margin` 控制重叠。
- **内容定宽**：固定列用固定宽度，自适应列用 `flex: 1`/`auto`。

---

### 两栏布局（左固定，右自适应）
- **Flex（推荐）**
  ```css
  .wrap { display: flex; }
  .left { width: 200px; }
  .right { flex: 1; }
  ```
  - 右侧自动填充剩余空间；结合 `gap`/`min-width` 控制细节。

- **浮动 + BFC**
  ```css
  .wrap { overflow: hidden; } /* 触发 BFC */
  .left { float: left; width: 200px; }
  .right { overflow: hidden; } /* 触发 BFC，自适应宽度 */
  ```
  - 右侧用 `overflow`/`display: flow-root` 等触发 BFC，避免与浮动重叠。

- **绝对定位 + `margin-left`**
  ```css
  .wrap { position: relative; }
  .left { position: absolute; left: 0; width: 200px; }
  .right { margin-left: 200px; }
  ```
  - 适合固定高度布局；注意 `right` 区域的宽度会随窗口变化。

---

### 三栏布局（左右固定，中间自适应）
- **Flex（推荐）**
  ```css
  .wrap { display: flex; }
  .left, .right { width: 200px; }
  .center { flex: 1; }
  ```
  - 代码直观；配合 `order` 可调整视觉顺序。

- **双浮动 + 中间 BFC**
  ```css
  .left { float: left; width: 200px; }
  .right { float: right; width: 200px; }
  .center { overflow: hidden; } /* BFC，避免被浮动覆盖 */
  ```
  - DOM 顺序常写成“左中右”，但浮动后视觉为“左中右”；记得清除浮动。

- **两侧绝对定位 + 中间 `margin`**
  ```css
  .wrap { position: relative; }
  .left { position: absolute; left: 0; width: 200px; }
  .right { position: absolute; right: 0; width: 200px; }
  .center { margin: 0 200px; }
  ```
  - 中间自动填满剩余空间；适合内容高度明确的场景。

- **Grid**
  ```css
  .wrap {
    display: grid;
    grid-template-columns: 200px 1fr 200px;
  }
  ```
  - 可进一步用 `grid-template-areas`/`minmax` 实现复杂布局。

### 面试提示
- 描述选择依据：设备兼容、响应式需求、元素高度是否自适应。
- 给出代码思路 + 关键属性，说明如何防止浮动塌陷或列重叠。
- 提及潜在问题：`flex` 在 IE9- 不支持、浮动需要清除、绝对定位脱离文档流等。
